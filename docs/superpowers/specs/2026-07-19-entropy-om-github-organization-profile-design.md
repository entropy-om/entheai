# entropy.om GitHub Organization Profile Design

**Date:** 2026-07-19
**Status:** Approved for implementation

## Purpose

Position `entropy.om` as the umbrella organization for entheai and the broader open, sovereign, local-first AI ecosystem. The profile should connect the ecosystem's philosophical identity—"entropy is the source"—with concrete work on agents, orchestration, and memory.

## Public profile

- **Organization name:** `entropy.om`
- **Description:** `Building sovereign intelligence in the open — local-first agents, memory, orchestration, and tools for minds that fan out.`
- **Website:** `https://github.com/entropy-om/entheai` until `https://entropy.om` has a DNS record and a live public site
- **Location:** unset
- **Public email:** unset
- **Social accounts:** unset

Keeping location, email, and social fields unset avoids exposing personal information that is not part of the public organization identity.

## Avatar

The avatar is an original, image-generated **broken-orbit** mark:

- a single warm amber seed at the center;
- electric-cyan arcs breaking and fanning outward;
- an obsidian-black field;
- a minimal, high-contrast silhouette legible at small sizes;
- composition safe for GitHub's circular crop;
- no letters, words, watermark, third-party mark, mockup, or 3D presentation.

The mark represents entropy becoming organized intelligence: one source, many trajectories, and a coherent center.

### Generation prompt

```text
Use case: logo-brand
Asset type: GitHub organization avatar
Primary request: Create an original abstract broken-orbit mark for entropy.om. A single warm amber point of light sits at the exact center while three to five electric-cyan arcs break open and fan outward, suggesting entropy becoming organized intelligence and one mind branching into many trajectories.
Scene/backdrop: deep uniform obsidian-black square field
Style/medium: minimal vector-like logo mark with flat geometric forms and a restrained luminous edge
Composition/framing: centered, symmetrical overall balance with subtle asymmetry in the broken arcs; strong silhouette; generous padding; all important details inside a circular safe area; legible at 32 pixels
Color palette: obsidian black, electric cyan, one warm amber-white center point
Constraints: icon only; original design; high contrast; clean edges; no letters; no words; no watermark; no mockup; no border; no unrelated symbols; no third-party marks
Avoid: dense networks, tiny nodes, photorealism, decorative clutter, excessive glow, lens flare, gradients that obscure the silhouette
```

## Implementation

1. Generate the square avatar with OpenAI image generation using the approved prompt.
2. Inspect the result at full size and at a 32-pixel preview.
3. If necessary, make one targeted image-generation revision to improve crop safety or small-size legibility.
4. Upload the selected avatar through GitHub organization settings.
5. Apply the approved name, description, and temporary website URL.
6. Re-read the public organization profile and verify the avatar, text, and link.

## Validation and recovery

- The description must match the approved copy exactly.
- The website must resolve to the entheai repository while `entropy.om` remains unavailable.
- The avatar must remain recognizable within GitHub's circular crop and at small sizes.
- Personal contact and location fields must remain unset.
- If the avatar upload or profile update fails, leave the previous field value intact and report the failing field rather than making unrelated account changes.

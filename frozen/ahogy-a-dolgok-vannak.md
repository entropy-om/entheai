**AHOGY A DOLGOK VANNAK** — *as things are*. Nothing more, nothing less, just how it should be.

The root creed under every other rule in this directory. Applied:

- **Nothing more:** never report a state better than reality. No fake liveness, no silent pass, no claim without empirical evidence (a seal, an exit code, a raw log). If something was skipped, say *skipped* and why.
- **Nothing less:** never hide a state worse than reality. Failures are ingested, not discarded; stale data says *stale*; an unverifiable merge stays unmerged and says so.
- **Just how it should be:** the report and reality are the same object. When they drift, reality wins and the report changes — never the other way around.

The idea

Plan once. Build in parallel.

The orchestrator decomposes the task and dispatches model‑matched sub‑agents into isolated worktrees. Nothing merges until it's verified.

Capabilities

Everything runs on one Mac.

◈
Tiered hybrid brain
A cloud orchestrator plans; local and cloud models execute — each task routed to the model that fits it.
⑃
Fan-out orchestration
Sub-agents run in parallel git worktrees and merge back only after their tests pass.
▤
Visual TUI
Animated shader backgrounds and a live 3D codebase graph, right in your terminal.
◱
Compounding memory
Five memory namespaces with auto-compaction — it remembers your codebase and your preferences.
⧉
Extensible
Skills, plugins, and MCP servers snap in. Bring your own tools and providers.
✦
Self-improving
Learns from every run and personalizes to how you work, over time.
